## Stack


- LARAVEL 10
- PHP 8.1
- Mysql 5.7
- APACHE2


## Installation

Add in .env file this params

ADMIN_EMAIL= your email

ADMIN_PASSWORD= your password




``` bash

composer install
php artisan migrate
php artisan db:seed

```

Site Url: 

http://127.0.0.1/

Admin Panel: 

http://127.0.0.1/admin
